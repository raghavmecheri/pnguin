pnguin.api.io.read_csv_to_data(path: str, delimiter: str = ',', headers: list = [])¶

A zero-dependancy helper method to read a csv file

Given the path to a csv file, read data row-wise. This data may be later converted to a dict of lists if needed (column-wise).

Parameters

  • path (str) – Path to csv file

  • delimiter (str, optional) – Delimiter to split the rows by. Defaults to ‘,’

  • headers – (list, optional): Given header list for a csv file. Defaults to an empty list, which results in the first row being used as a header.

Returns

A list of dictionary values (list of rows) representing the file being read

Given a subscript, which may be a slice, a string, or an index, fetch the required items. This function assumes that the data-subscript pairs have already been checked using is_valid_index.

  • k (any) – A subscript that may be a slice, an int, or a string

pnguin.api.utils.is_valid_index(idx, axis)¶

Given a subscript and an axis, check if it’s valid

Different DataFrames have different compatible axes. Some are supported via conversion, and some are incompatible. Filter through the possible combinations, and throw the required errors or print the required warnings.

Parameters

  • idx (any) – The subscript being passed

  • axis (Axis) – An enum signifying the Axis of the DataFrame.

Returns

An indicator of index validity str: A message, if any should be printed Axis: The target_axis that the data would need to be in

Return type

bool
